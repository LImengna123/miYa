##蜜芽介绍

>蜜芽的前身蜜芽宝贝[1]  ，是中国首家进口母婴品牌限时特卖商城。由全职妈妈刘楠于2011年创立，希望创造简单、放心、有趣的母婴用品购物体验。“母婴品牌限时特卖”是指每天在网站推荐热门的进口母婴品牌，以低于市场价的折扣力度进行特卖    —— [蜜芽](https://baike.baidu.com/item/%E8%9C%9C%E8%8A%BD)
